# FTP-Server

A simple FTP server\client implemented with Python sockets. 

--------

## Requirements

In your virtual environment run:

```
pip install -r requirements.txt
```

## Tutorial

Running the server
```
python server.py
```

Running a client

```{bash}
python server.py
```

----

You can read more by looking at the [docs](https://rkanehisa.github.io/FTP-Server/)!
